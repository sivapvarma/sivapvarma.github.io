Hello World!

Posted on Aug 12, 2015 in Journal

For a long time, I wanted to have a blog to document stuff so that I could refer back to them later. Back then in 2010, I looked at Blogger, Wordpress and Tumblr. I did not like them as I had very limited control over the design.

Static Site Generators and Github hosting means I can have all the freedom at zero cost. There is no dearth of Static site generators these days. I have tried Pelican, Jekyll, Hakyll and feel pelican suits me well.

I liked Jekyll a lot, especially the fact that I can directly push markdown to my repo rather than the html generated from markdown. However, I never used Ruby. So I did not want to install Ruby, Jekyll and its many dependencies on my computer just for blogging.

Hakyll put me in a situation where I was not able to reproduce the setup from one computer to another. Also I am not as comfortable with Haskell as I am with Python. Pelican’s stability also should help me concentrate on writing the content. Eventually I came back to Pelican which I tried first.
